#Todo List Practice using plain vanilla JavaScript

1. Add Task
2. Mark Complete Task
3. Revert Completed Task to Incompleted Task
4. Delete Task
5. Filter Tasks

#Setup

1. **Clone repo**
2. **Install packages**
    ```
    npm install
    ```
3. **Start server**
    ```
    npm start
    ```
4. **Open localhost on port 8080**
    ```
    http://localhost:8080/
    ```
5. **Run Gulp**
   ```
   gulp or gulp watch
   ```
